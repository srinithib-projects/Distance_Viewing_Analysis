Distance Viewing Analysis of Historical Postcards Erlangen

This project analyzes visual patterns in historical postcards of Erlangen using distance viewing techniques. Instead of studying individual images, the project examines the overall visual characteristics of a postcard collection to identify trends over time.

Project Overview:  
The dataset consists of 50 digitized postcards from Erlangen (1897–1921) collected from Wikimedia Commons. The postcards depict various scenes such as cityscapes, land marks, railway stations, markets, and public spaces.
The aim of the study is to examine how colour and tonal properties changed over time.

Methodology:  
The analysis was performed using Distant Viewing Explorer (DVE). The following visual features were extracted:  
Brightness – overall lightness of the image  
Saturation – intensity of colours  
Edge Score – level of visual detail and complexity  
Hue Distribution – proportion of colours such as red, yellow, green, blue, etc.  

The extracted data was analyzed to identify temporal patterns across the postcard collection.

Key Findings:  
Early postcards are mostly monochrome or lightly tinted.  
Over time, postcards become brighter and more colourful.  
Later postcards show greater colour diversity and visual variation.  
Improvements in printing technologies likely contributed to these changes.  

Academic Context:  
This project was completed for the seminar “Pattern Recognition in the Humanities” at Friedrich-Alexander-Universität Erlangen–Nürnberg (FAU).
